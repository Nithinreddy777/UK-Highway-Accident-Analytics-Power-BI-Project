# 🚗 UK Highway Accident Analytics | Power BI Project

## 📌 Overview
The UK Highway Accident Analytics dashboard is an interactive Power BI project created to analyze road accident data across the United Kingdom. This dashboard focuses on accident trends, road conditions, weather impact, accident severity, and vehicle involvement to generate meaningful business insights using data visualization techniques.

---

## 🚀 Project Objectives
- Analyze highway accident patterns
- Identify accident trends by month
- Compare accidents across different road types
- Study weather and light condition impacts
- Build an interactive and user-friendly dashboard
- Practice real-world business intelligence reporting

---

## 🛠 Tools & Technologies Used
- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation
- Data Visualization

---

## 📊 Dashboard Features
### ✔ KPIs
- Average Speed
- Count of Vehicles
- Total Accidents
- Total Casualties

### ✔ Visualizations
- Count of Accidents by Month
- Accidents by Road Type
- Accidents by Weather Conditions
- Accident Severity Distribution

### ✔ Interactive Filters
- Date Filter
- Road Type Filter
- Light Conditions Filter
- Junction Filter
- District Filter

---

## 📈 Key Insights
- October recorded the highest number of accidents.
- Single carriageway roads had the highest accident count.
- Most accidents were categorized as slight severity.
- Weather conditions significantly impacted accident frequency.
- Accident trends fluctuated throughout the year.
